# Portfolio — Aguwa Chinedu (Civil & Structural Engineer)

## Stack
- **Next.js 16** (App Router) — read `node_modules/next/dist/docs/` before touching routing or config
- **React 19**, TypeScript 5, Tailwind CSS 4
- **Three.js** — 3D construction scene (`ConstructionCanvas.tsx`)
- **GSAP + ScrollTrigger** — drives the 3D animation scrub only; do not use for UI
- **Framer Motion** — all UI animations (hover, reveal, page transitions)
- Package manager: **pnpm** (`pnpm dev`, `pnpm lint`, `pnpm build`)

## Design tokens (globals.css)
| Token | Value | Usage |
|---|---|---|
| `--background` | `#000000` | page background |
| `--foreground` | `#f5f0eb` | primary text |
| `--muted` | `#8a7a6a` | secondary text |
| `--accent` | `#ff6600` | construction orange — buttons, highlights |
| `--line` | `rgba(255,102,0,0.18)` | borders/dividers |

Use `text-accent`, `bg-accent`, `border-line` etc. via Tailwind. Never hardcode `#ff6600` in UI components — use `var(--accent)` or the Tailwind token.

## Animation rules
- **GSAP ScrollTrigger** — only in `ConstructionCanvas.tsx` and `PhaseRail.tsx` for scroll-linked 3D/canvas work. `scrub: 1`.
- **Framer Motion** — all other animation. Use `whileInView` + `viewport={{ once: true }}` for scroll reveals. Use spring (`type: "spring"`) for interactive hover/tap. Use `AnimatePresence` for mount/unmount.
- **`Reveal` component** — wrap any section content that should fade in on scroll. Accepts `delay` (ms).
- Never call `setState` synchronously inside a `useEffect` body — wrap in `setTimeout(() => ..., 0)` (linter enforces this).

## Three.js scene — ConstructionCanvas.tsx
- **Road**: z = 10 → 22 (centre z = 16, depth = 12). Nothing may have a footprint overlapping this range.
- **Buildings**: near side z ≤ 4; far side z ≥ 27. Use `makeBgBuilding(ox, oz, bays, baySize, depth, floors, floorH)` — always `floorH = 3.0`.
- **Camera** (at scroll progress `p`): azimuth orbits from π/3 → π/7.2, dist 62 → 33, height 30 → 20, fov 48 → 41. Starts wide showing site ground + surroundings, pulls in as building rises.
- **LED sign**: do not redesign without explicit instruction — last good state is in git.

## PhaseRail (level staff)
- 12 alternating **orange / white** bands (2 per phase), E-pattern marks, yellow crosshair climbs bottom → top as scroll progresses.
- Maps 6 phases: Site prep → Foundation → Frame → Slabs & walls → Facade → Topped out.
